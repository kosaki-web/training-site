## 🎨 コンセプト
**「親しみやすさと直感的な操作性の両立」**
- 普段インターネットを頻繁に利用しない保護者の方でも、迷わず情報に辿り着けるよう、視認性の高いレイアウトと大きなボタン配置を徹底しました。
- 企業情報は架空のものに差し替え、ポートフォリオ用としてブラッシュアップしています。

## ✨ 実装・こだわったポイント
### 1. 高度なスクロール制御（CTAボタン・TOPボタン）
- **UXの最適化**: ファーストビューでは没入感を優先してCTAボタン（申し込みボタン）を非表示にし、スクロールに応じて自然に出現するようJSで制御しました。
- **緻密な配置制御**: ページ下部において、CTAボタンがフッターエリアと重ならないよう、特定のポイントで固定を解除して停止させる実装を行いました。

### 2. インタラクティブな要素の実装
- **カルーセルパネル**: 「お客様の声」セクションにスライド機能を実装し、情報の整理とサイトへの動きを両立。
- **アコーディオンメニュー**: `details`タグを活用し、視覚的にスッキリさせつつ、必要な情報をすぐに確認できるQ&Aセクションを構築。
- **モバイルファースト**: スマートフォンでの閲覧を主軸に置き、親指で押しやすいボタンサイズやハンバーガーメニューを設計。

### 3. AI技術の戦略的活用
- **効率的な開発**: 指示書に基づいた要件定義の壁打ちや、複雑なCSS Grid/JS制御のベースコード生成にGeminiを活用。AIをパートナーとして利用することで、短期間で高機能なサイト構造を実現しました。

## 🛠 使用ツール・技術

- **デザイン**: Figma, Photoshop
- **エディタ**: VS Code
- **AI活用**: Gemini (コード生成・構成案・画像生成指示)
- **外部リソース**: Google Fonts (Noto Sans JP / Train One等), destyle.css

## 📝 制作の背景と課題解決
職業訓練校での「実案件想定課題」として制作。
「指示書からターゲットの悩みを読み解く」というステップを重視しました。例えば、保護者の「初期費用や機材への不安」という悩みに応えるため、Q&Aや無料レンタルのアイコンを強調するなどの工夫を施しています。
